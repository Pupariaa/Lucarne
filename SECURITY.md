# Security policy

## Supported versions

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |

## Reporting a vulnerability

Do **not** open a public issue for security-sensitive reports.

Send details to the repository owner via GitHub private security advisory:

[Report a vulnerability](https://github.com/Pupariaa/Lucarne/security/advisories/new)

Include steps to reproduce, affected components (firmware, Studio, website), and impact if known.
